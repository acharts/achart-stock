## Usage

  1. 引入cdn上的代码 
  ```js
    <script src="http://g.tbcdn.cn/bui/acharts/1.0.12/astock-min.js"></script>
  ```

  2. 使用,引入astock文件后，会在window上增加AStock变量代表AStock控件类

  ```js
  var stock = new AStock({
    //configs
  });
  ```
